# 09 指标选取方案（数据下载操作蓝本）

> 本文件是你"定指标、下数据"阶段的操作蓝本。看完你应当能：①明确要下哪些变量；②每个变量去哪下、下什么口径；③下下来后怎么变换成可建模的形式。**建议把第五节的"下载清单总表"打印出来逐项打勾。**

---

## 〇、总体决策（先定框架，再下数据）

| 决策项 | 方案 | 理由 |
|--------|------|------|
| **数据频率** | 月度（monthly） | 油价→通胀是中短期传导，季度太粗、日度太噪 |
| **样本期** | 2000:01 — 2024:12（约 300 个月） | 覆盖 2008 危机、2014 崩盘、2020 负油价、2022 飙升四次大波动，利于识别；新兴市场月度数据 2000 后较齐 |
| **国家范围** | G20 的 **19 个国家成员** | 欧盟、非盟为区域组织不单列；名单见 `05` |
| **数据三大类** | ①国别宏观面板 ②全球油市块 ③国家特征 | 分别用于：QFAVAR 因子 / 冲击识别 / 第二阶段回归 |
| **建模对象** | 各变量经**平稳化 + 标准化**后进入模型 | 见第四节 |

> **样本期备选**：若新兴市场（尤其阿根廷、印尼）早期月度数据缺口大，可备选 **2005:01** 或 **2010:01** 起点，在稳健性中说明。**先按 2000 下，缺多少再决定。**

## 一、第一类：国别宏观变量（进入 QFAVAR 的面板，每国一套）

QFAVAR 对"每个变量、每个分位数"提取一组跨国因子。变量越多，状态 VAR 越大。原文用 5 个变量；本文建议**核心 4 个**（务必下），外加**可选扩展**（做深化模块时再下）。

### 1.1 核心变量（必下，4 个）★

| 序号 | 变量 | 英文 | 选取理由 | 建模变换 | 单位 |
|------|------|------|----------|----------|------|
| ① | **消费者价格指数 → 通胀率** | CPI inflation | **本文核心被解释变量**，通胀分布是研究对象 | 见下方"通胀口径" | % |
| ② | **工业生产指数** | Industrial Production (IP) | 代理实体经济活动；油价冲击同时打击产出，需纳入 VAR | 同比增长率，或对数差分×100 | % |
| ③ | **短期/政策利率** | short-term / policy interest rate | 货币政策反应渠道，决定二轮效应与通胀持续性 | 水平值（已平稳）；或一阶差分作稳健 | % |
| ④ | **名义有效汇率** | Nominal Effective Exchange Rate (NEER) | 油以美元计价，本币升贬值放大/缓冲输入性通胀；**截面异质性关键来源** | 对数差分×100（升值为正） | % |

**通胀口径（重要，建议这样定）**：
- **主口径**：同比通胀 $\pi_t = 100\times[\ln(CPI_t)-\ln(CPI_{t-12})]$。优点：平滑、可解释（"第 90 百分位的同比通胀"直观）、是油价—通胀文献标准。
- **稳健口径**：季调环比年化 $\pi_t = 1200\times[\ln(CPI^{SA}_t)-\ln(CPI^{SA}_{t-1})]$。优点：捕捉更快动态；缺点：噪声大。
- 两者都跑，正文报主口径，稳健性报另一口径。**注意**：同比口径有 12 个月重叠的移动平均结构，靠 VAR 滞后阶吸收即可。
- 也可考虑直接对 **CPI 指数**建模（如原文做法），但本文研究"通胀"，建议用通胀率，解释更自然。

### 1.2 可选扩展变量（做深化模块时再下）

| 变量 | 用途 | 数据源 |
|------|------|--------|
| **核心 CPI**（剔除食品能源） | 看油价是否传导到**核心通胀**（二轮效应，央行最关心） | OECD、各国统计局 |
| **能源/食品 CPI 分项** | CPI 成分分解模块 | OECD、Eurostat、各国统计局 |
| **生产者价格指数 PPI** | 上游成本传导（油→PPI→CPI） | OECD、IMF IFS |
| **通胀预期** | 预期渠道 | 调查数据（OECD、密歇根、央行）、可得性差 |
| **股指 / 金融压力指数** | 金融渠道（仿原文 CLIFS） | OECD、各国交易所、OFR/ECB CLIFS |
| **失业率 / 零售销售** | 实体活动补充 | OECD |

> **建议**：基准模型用核心 4 变量；"核心 CPI"强烈建议一并下（二轮效应模块性价比最高）。

## 二、第二类：全球油市块 $\boldsymbol{g}_t$（用于识别三类油价冲击）★★

这是识别的命脉。要支持 Kilian(2009)/Kilian-Murphy(2014) 的三冲击分解，$\boldsymbol{g}_t$ 至少含下列前三项，**强烈建议加第四项（库存）**。

| 变量 | 英文 | 主要识别哪类冲击 | 建模变换 | 推荐数据源（基准） |
|------|------|------------------|----------|--------------------|
| **实际油价** | real price of oil | 三类冲击共同的价格变量 | 名义油价÷美国CPI，取对数（或对数差分） | **Brent**：FRED `MCOILBRENTEU` |
| **全球石油产量** | world crude oil production | 油价供给冲击 | 对数差分×100（月度产量增速） | **EIA** 国际数据（世界原油/液体燃料产量，月度） |
| **全球经济活动指数** | global real economic activity | 全球总需求冲击 | 多为已平稳指数，直接用（或标准化） | **Kilian 指数**：FRED `IGREA`（Dallas Fed） |
| **全球石油库存** | global crude oil inventories | 区分油价特定需求 vs 总需求 | 对数差分×100 | OECD 石油库存；或 EIA 美国库存按 OECD/US 比例放大（Kilian-Murphy 代理） |

**油价指标的三套（基准 + 稳健）**：
- 基准：**Brent**（最接近全球定价）。
- 稳健 1：**WTI**（FRED `MCOILWTICO`）。
- 稳健 2：**美国炼厂收购成本 RAC**（EIA，Kilian 原文用）。
- 三者都要**转成实际油价**（÷美国 CPI，FRED `CPIAUCSL`），否则含通胀成分污染识别。

**全球经济活动的两套（基准 + 稳健）**：
- 基准：**Kilian 全球实际经济活动指数**（FRED `IGREA` / Dallas Fed 网站 CSV）。
- 稳健：**Baumeister-Hamilton 世界工业生产指数**（Baumeister 个人主页 datasets）。

**外部工具变量（稳健性识别，非 $g_t$ 成员）**：
- **Känzig (2021) OPEC 油价供给新闻冲击工具序列** —— GitHub `dkaenzig/oilsupplynews`（定期更新）。用于代理 SVAR 稳健性，独立验证供给冲击。

> **可选附加全球控制**：全球供应链压力指数 GSCPI（NY Fed）、全球经济政策不确定性 GEPU（policyuncertainty.com）。本文聚焦油价，可不放入主识别，仅作稳健性控制变量。

## 四、统一的变量变换与数据清洗规则

下载到的多是"原始指数/水平"，进入模型前要统一处理。**对每个序列按下表操作**：

| 变量 | 季调 | 平稳化变换 | 进模型前是否标准化 |
|------|------|------------|--------------------|
| CPI → 通胀 | CPI 需季调（用 SA 版或自行 X-13） | 同比或季调环比年化对数差分 | 是（z-score） |
| 工业生产 IP | 季调 | 同比或对数差分×100 | 是 |
| 短期利率 | 否 | 水平值（必要时一阶差分） | 是 |
| NEER | 一般无需 | 对数差分×100 | 是 |
| 实际油价 | 否 | 对数（水平）或对数差分 | 是 |
| 全球产量 | 季调 | 对数差分×100 | 是 |
| 全球活动指数 | 已处理 | 直接用 | 是 |
| 石油库存 | 季调 | 对数差分×100 | 是 |

**清洗五步**：
1. **统一频率**：全部月度；季度序列插值或剔除该变量。
2. **季节调整**：用各源 SA 版本；若只有 NSA，用 X-13ARIMA-SEATS。
3. **平稳性检验**：对每个变换后序列做 ADF / PP / KPSS，确认平稳（写进附录）。
4. **异常值/缺失**：极端通胀（阿根廷、近年土耳其）做缩尾（winsorize）或单列；短缺口插值，长缺口缩短共同样本。
5. **标准化**：进入因子提取前，每个序列去均值、除以标准差（z-score），使量纲可比；保存均值与标准差以便结果还原。

## 六、数据整理的最终形态（给代码用）

最终要整理成两个对齐的矩阵（行 = 月份 $T$，所有序列已变换+标准化）：

1. **国别数据矩阵 `Y`**：列按"**变量大块 → 块内国家**"排列，例如
   `[CPI_AR, CPI_AU, …, CPI_US | IP_AR, …, IP_US | RATE_… | NEER_…]`。
   **顺序很重要**：QFAVAR 的载荷矩阵是"变量分块对角"，列顺序必须与"变量专属因子"的分块结构一致（同一变量的各国列连续排在一起）。
2. **全球油市矩阵 `g`**：`[实际油价, 全球产量, 全球活动, 石油库存]`。
3. 另存：分位数向量 `quantiles = [0.10, 0.50, 0.90]`、滞后阶 `p=2`、各序列的均值/标准差（还原用）、国家与变量标签。

**文件命名建议**（放 `data/`）：
- `g20_cpi_monthly.csv`、`g20_ip_monthly.csv`、`g20_rate_monthly.csv`、`g20_neer_monthly.csv`
- `global_oil_block.csv`、`kanzig_instrument.csv`
- `Y_matrix.csv`、`g_matrix.csv`（清洗后供代码直接读取）
