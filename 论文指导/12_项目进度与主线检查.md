## 一、北极星：一句话主线（所有工作都必须服务于它）

> **不同来源的国际油价冲击，如何不对称地改变各国通货膨胀分布的形状（尤其是高通胀的尾部风险），以及这种跨国差异由哪些结构性因素决定。**

题目：**国际油价冲击对通货膨胀的异质性影响——基于 QFAVAR 模型的多国研究**。
"异质性"= **分布异质性**（分位数间）+ **截面异质性**（国家间）。

---

## 二、全文逻辑链（through-line）

```
① 识别三类油价冲击（供给/总需求/油价特定需求）
        ↓
② 分位数脉冲响应：油价冲击对通胀 10/50/90 分位数的不对称影响   ← 分布异质性（核心结果一）
        ↓
③ 国别投影：把因子响应投影回各国，比较传导差异              ← 截面异质性（核心结果二）
        ↓
④ 第二阶段回归：用国家结构特征解释"为什么各国不同"          ← 文章深度所在
        ↓
⑤ 历史分解 + 情景分析：归因 2021–2023 通胀大潮 + 通胀在险      ← 政策落点
```

三大核心产出：**分位数脉冲响应图 / 国别异质性 / 第二阶段"解释异质性"**。其余（拟合、方差分解、稳健性）都是支撑。

## 四、当前阶段方案（数据阶段·已定稿）

- **数据**：16 国、248 个月、9 变量；来源/口径/变换见 `11`；中国 IP 1-2 月置缺失、日本利率前 8 月赋 0、命名/顺序/日期戳已统一。
- **模型**：QFAVAR；分位数 **10/50/90**（稳健加 5/25/75/95）；滞后 **p=2**；两步法为主 + 一步法做结构识别。
- **识别（已锁定，以 `07` 为准）**：
  - 基准 = **符号约束 + 供给弹性约束**（只约束油市块 + 中位数因子；尾部分位数因子全自由）；
  - **GSCPI、GEPU 仅作控制变量**；
  - 稳健性 = Cholesky 递归 / 叙事约束 / Känzig 工具 / 补库存后 Kilian-Murphy。

## 六、主线检查：有没有跑偏？

### 6.1 各组成是否服务主线（逐项体检）

| 组成 | 是否服务主线 | 说明 |
|------|--------------|------|
| CPI（通胀）| ✅ 核心 | 研究对象本身（通胀分布） |
| 油市块（产量/IGREA/油价）| ✅ 核心 | 识别三类油价冲击 |
| 工业生产、利率、NEER | ✅ 支撑 | 实体/政策/汇率渠道的内生变量 |
| GSCPI、GEPU | ✅ 支撑 | 控制混淆，净化油价识别 |
| QFAVAR + 分位数 | ✅ 核心 | 唯一能同时刻画两类异质性的工具 |
| 第二阶段回归 | ✅ 深度 | 把"发现差异"升级为"解释差异" |

**结论：所有组成都对齐主线，方向没有偏。**

### 6.2 需要警惕的"跑偏风险"（提醒）

1. **不要让数据细节喧宾夺主。** 我们在中国 IP、日本利率、NEER 口径、季节调整上花了很多轮——这些是**必要的地基**，但**不是论文的贡献**。论文的价值在"分布 + 跨国异质性"，不在数据清洗。**下一步重心必须转向建模与异质性分析。**
2. **守住"尾部/通胀在险"这条命脉。** 文章的卖点是"油价主要推高通胀**右尾/尾部风险**"，不是又一篇"油价→通胀均值"。出结果时，**分位数发散、尾部反应**必须是叙事中心。
3. **IP 是配角，CPI 是主角。** 中国工业增加值等口径瑕疵已妥善处理，不必再纠结——它不会动摇主结论。
4. **别过度工程化。** 欧元区利率重复、GEPU 非平稳等都是次要项，放稳健性即可，不要再投入过多精力。
5. **第二阶段和 2021-23 历史分解是"深度"和"政策"的来源，务必做。** 否则容易退化成"换数据跑别人模型"。

### 6.3 一句话体检结论
> **没有跑偏，方向稳。** 数据阶段已扎实收尾；现在要做的是**把重心从"数据完善"切换到"模型结果 + 异质性解释"**，让论文的真正贡献浮出水面。
