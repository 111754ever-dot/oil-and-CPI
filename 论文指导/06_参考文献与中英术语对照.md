## 第二部分：中英学术术语对照表（写作 + 阅读随时查）

### 模型与方法

| 中文 | English | 简短解释 |
|------|---------|----------|
| 分位数因子增广向量自回归 | Quantile Factor-Augmented VAR (QFAVAR) | 本文核心模型 |
| 向量自回归 | Vector Autoregression (VAR) | 多变量时间序列联合建模 |
| 因子增广 VAR | Factor-Augmented VAR (FAVAR) | 用因子压缩信息的 VAR |
| 动态因子模型 | Dynamic Factor Model (DFM) | 从大量变量提取共同因子 |
| 分位数因子模型 | Quantile Factor Model (QFM) | 因子可移动分布不同分位数 |
| 分位数回归 | Quantile Regression | 估计条件分位数而非均值 |
| 状态空间模型 | State-Space Model | 测量方程 + 状态方程 |
| 测量方程 / 状态方程 | Measurement / State (Transition) Equation | 因子→数据 / 因子的动态 |
| 因子载荷 | Factor Loading | 变量对因子的敏感度（λ） |
| 非对称拉普拉斯分布 | Asymmetric Laplace (AL) Distribution | 连接分位数回归与似然 |
| 贝叶斯推断 | Bayesian Inference | 用先验 + 数据得后验 |
| 马尔可夫链蒙特卡洛 | Markov Chain Monte Carlo (MCMC) | 从后验抽样的算法 |
| 吉布斯抽样 | Gibbs Sampler | 一种 MCMC 方法 |
| 马蹄先验 | Horseshoe Prior | 自动收缩、稀疏化的先验 |
| 收缩 / 正则化 | Shrinkage / Regularization | 把不重要参数压向 0 |
| 变分贝叶斯 | Variational Bayes | 快速近似后验（两步法） |
| 两步法 | Two-step procedure | 先估因子、再估 VAR |

| 零约束 / 排他性约束 | Zero / Exclusion Restrictions | 某些当期反应设为 0 |
| 脉冲响应函数 | Impulse Response Function (IRF) | 冲击后变量随时间的反应 |
| 广义脉冲响应 | Generalized IRF (GIRF) | 不依赖排序的 IRF |

| 情景分析 | Scenario Analysis | 用尾部反应作"最坏情景" |
| 识别 | Identification | 从相关中分离出因果冲击 |
