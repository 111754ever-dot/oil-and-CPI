## 一、B2 做了什么

把作者的 GIRF（广义脉冲响应）主程序适配到我们的数据：**16 国、4 个宏观变量、5 个全球变量**。这一版能跑出：分位数因子、广义脉冲响应、方差分解、国别投影。**符号约束识别不在本步（那是 B3）。**

维度对照：`n=64`(4×16)、`r=4`、`ng=5`、`nq=3`、状态维 `r*nq+ng=17`、`k=34`。

## 三、★ 故意"没有改"的地方（防呆，最容易误伤）

代码里还有一些"看起来是 9 或 4"但**不是国家数**的，**必须保持原样**：

| 位置 | 内容 | 为什么不能改 |
|------|------|--------------|
| 第 36 行 | `quant = [.1, .5, .9]` | `.9` 是**分位数 0.9**，不是国家数 |
| 第 240/252 行 | `9*ones(i-1,1)` (出现 2 次) | `9` 是 VAR 递归系数的**先验方差超参数**，非国家数 |
| 第 229 行 | `SVRW(...,4)` | `4` 是随机波动函数的参数(且 var_sv=0 不启用) |
| 第 502 行 | `mean(fevd_save,4)` | `4` 是**求平均的维度**(第 4 维=MCMC 抽样) |
| 多处 | `0.9290 0.6940 ...`、`10%/50%/90%` | 绘图颜色 / 分位数图例文字 |

> 脚本已确认这些**全部保持原样**（防呆检查通过、无残留 country-9）。

## 四、维度追踪验证（用 n=64,r=4,ng=5 手算，确认索引正确）

- 因子提取 `ifac=1..4` 取列 → **1-16 / 17-32 / 33-48 / 49-64**，正好对应 Y 的 CPI/IP/RATE/NEER 四块 ✅
- `floor((i-1)/16+1)`：i=1..16→块1(CPI)、17..32→块2(IP)、33..48→块3、49..64→块4 ✅
- 状态维 `r*nq+ng = 17`；每方程参数 `k = 17×2 = 34` ✅
- FEVD 全局块索引 `nq*r+1:end = 13..17` → 正好 5 个全局变量 ✅

## 五、怎么运行 & 自查（阶段 C）

**文件摆放**（同一文件夹）：
```
你的运行目录/
  ├─ QFAVAR_GIRF_oil.m      (本文件)
  ├─ load_data_oil.m        (读数函数)
  ├─ all_data.xlsx          (你的数据; 或放 data/ 子目录)
  ├─ functions/             (作者的函数库: VBQFA/FFBS/... 全部)
  └─ data/                  (若把 all_data.xlsx 放这里)
```

**运行前必做（D4：先调试后定稿）**：把第 50-51 行的 MCMC 次数**先改小**：
```matlab
nsave = 2000;   nburn = 500;   nthin = 20;   % 调试用(几分钟~十几分钟)
% 结果正常后再改回 nsave=20000~50000 出定稿
```

**跑通后看什么**：①分位数因子图(10/50/90 三线是否发散)；②因子对各全局变量的广义脉冲响应；③FEVD；④国别投影。

## 六、需要你知道的"观察项"（非错误）

1. **文本型数字（你已决定暂不处理）**：若 `readtable` 读 `all_data.xlsx` 时报错(某列被当字符串)，把报错发我，一分钟可修（换纯数值版或加 `tbl2num`）。
2. **图例里下划线**：`oil_prod`、`loil_price` 在部分标题里会把 `_` 后当下标显示（纯粹好看问题，不影响结果；FEVD 标签我已用连字符规避）。
3. **国别图很密**：国别投影是 4×16=64 个小图，密但可用；后续可按需拆分美化。
